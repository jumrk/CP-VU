# 製品紹介ランディングページ

Next.js、Tailwind CSS、JavaScriptを使用した製品紹介のランディングページです。

## セットアップ

依存関係をインストール:

```bash
npm install
```

## 開発サーバーの起動

開発モードでアプリケーションを起動:

```bash
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いて結果を確認してください。

## ビルド

本番用にビルド:

```bash
npm run build
```

## スタート

本番モードでアプリケーションを起動:

```bash
npm start
```

## 技術スタック

- **Next.js 14** - React フレームワーク
- **Tailwind CSS** - ユーティリティファーストのCSSフレームワーク
- **JavaScript** - プログラミング言語
